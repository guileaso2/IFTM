Apresentação básica de cada Disciplina

PROJETO DE WEBSITE ESTÁTICO

É aqui que tudo começa! Você aprende a estrutura (HTML) e a aparência (CSS) dos sites. O termo "estático" significa que o site não muda sozinho ou não busca informações de um banco de dados em tempo real; ele exibe exatamente o que você escreveu no código.

LÓGICA DE PROGRAMAÇÃO

A base de qualquer programador. Aqui você não aprende uma linguagem específica, mas sim como pensar para resolver problemas. Você aprende o que são variáveis, condições (se isso, faça aquilo) e repetições. É o "alfabeto" da programação.

REQUISITOS

Antes de programar, precisamos entender o que o cliente quer. Nesta matéria, você aprende a documentar as necessidades do projeto. É como fazer a "lista de compras" e a "planta baixa" de um prédio antes de começar a obra.

SISTEMAS OPERACIONAIS

Você estuda como o Windows, Linux ou macOS funcionam por "baixo do capô". Aprende como o computador gerencia a memória, o processador e os arquivos para que seus programas consigam rodar sem travar

FUNDAMENTOS DE SISTEMAS PARA INTERNET

Uma visão geral de como a rede mundial funciona. Você entende o que é um servidor, o que é um IP, como os dados viajam de um lado para o outro e como a arquitetura da web é montada.

EMPREENDEDORISMO

Nem tudo é código! Aqui você aprende sobre o mercado de trabalho, como criar novos negócios, identificar oportunidades na área de tecnologia e como transformar uma ideia de software em um produto real.
